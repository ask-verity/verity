# Verity Documentation

This directory contains conceptual documentation related to **Verity**,
a conversational analytics agent for business data.

These documents explain the ideas, terminology, and design principles
behind conversational analytics, without exposing production implementation details.

## Contents

- What conversational analytics is
- Why dashboards break down for ad-hoc questions
- High-level architecture concepts
- Example questions and interpretations

This documentation is intended for:
- Product understanding
- Technical context
- Public reference and discussion

Production code and internal implementation details are not included.
