# Example Questions

This directory contains example questions and interpretations
that illustrate how Verity approaches conversational analytics.

These examples are conceptual and do not include executable code.

## Example

**Question**  
“How did marketing spend impact revenue this month?”

**Interpretation**
- Metrics: revenue, marketing spend
- Time range: current month
- Comparison: previous month

**Expected Output**
- Combined chart showing spend vs revenue
- Explanation describing correlation and changes

The purpose of these examples is to show *how* questions
are interpreted, not *how* the system is implemented.
